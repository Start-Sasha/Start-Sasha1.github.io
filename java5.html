17 1. Какая разница между использованием document.getElementById и document.getElementsByClassName для доступа к элементам DOM в JavaScript? 
document.getElementById:
    - Возвращает один элемент по его уникальному идентификатору (id).
    - Если элемент с указанным id не найден, возвращает null.
    - Пример:
        let element = document.getElementById('myElement');

document.getElementsByClassName:
    - Возвращает коллекцию элементов (HTMLCollection), которые имеют указанный класс.
    - Если элементы не найдены, возвращает пустую коллекцию.
    - Пример:
        let elements = document.getElementsByClassName('myClass');

Разница:
    - getElementById работает с уникальными идентификаторами и возвращает один элемент.
    - getElementsByClassName работает с классами и возвращает коллекцию элементов.
18 2. Какой метод JavaScript подходит для выбора нескольких элементов с одним селектором, а не только первого? 
Для выбора нескольких элементов с одним селектором используется метод document.querySelectorAll.

document.querySelectorAll:
    - Возвращает NodeList всех элементов, соответствующих указанному CSS-селектору.
    - Пример:
        let elements = document.querySelectorAll('.myClass');
19 3. Как можно динамически создавать элементы веб-страницы с помощью JavaScript? 
Для динамического создания элементов используется метод document.createElement и методы для 
добавления элементов в DOM, такие как appendChild или insertBefore.

Пример:

// Создание элемента
let newElement = document.createElement('div');
newElement.textContent = 'Новый элемент';

// Добавление элемента в DOM
document.body.appendChild(newElement);

21 5. Какие методы JavaScript нужно использовать для установки кодировки символов и масштабирования страницы на мобильных устройствах?

- Масштабирование на мобильных устройствах:

22 6. Как можно создать таблицу соответствия между данными, используя цикл for и динамическое добавление строк через JavaScript? 
Пример:

let table = document.createElement('table');
table.border = '1';

for (let i = 1; i <= 5; i++) {
    let row = table.insertRow();
    let cell1 = row.insertCell();
    let cell2 = row.insertCell();
    cell1.textContent = i;
    cell2.textContent = i * i;
}

document.body.appendChild(table);
23 7. Как можно форматировать числовые значения перед их выводом на веб-страницу в JavaScript? 
Для форматирования чисел используются методы:
    - toFixed(n) — округляет число до n знаков после запятой.
    - toLocaleString() — форматирует число в соответствии с локалью.

Пример:

let number = 1234.5678;
console.log(number.toFixed(2)); // "1234.57"
console.log(number.toLocaleString()); // "1,234.568" (зависит от локали)
24 8. Какое преимущество имеет использование document.querySelector перед document.getElementById при выборе элементов на веб-странице? 
document.querySelector:
    - Позволяет использовать любой CSS-селектор.
    - Возвращает первый элемент, соответствующий селектору.
    - Пример:
        let element = document.querySelector('#myId .myClass');

document.getElementById:
    - Работает только с id.
    - Возвращает один элемент.

Преимущество:
querySelector более гибкий, так как поддерживает сложные селекторы.
25 9. Как можно изменить текстовое содержимое элемента на веб-странице при нажатии на кнопки "+" и "-" с помощью JavaScript? 
Пример:

<p>Значение: <span id="value">0</span></p>
<button id="increase">+</button>
<button id="decrease">-</button>

<script>
    let valueElement = document.getElementById('value');
    let increaseButton = document.getElementById('increase');
    let decreaseButton = document.getElementById('decrease');

    increaseButton.addEventListener('click', () => {
        valueElement.textContent = parseInt(valueElement.textContent) + 1;
    });

    decreaseButton.addEventListener('click', () => {
        valueElement.textContent = parseInt(valueElement.textContent) - 1;
    });
</script>
26 10. Как можно организовать динамическое обновление содержимого таблицы без перезагрузки страницы при помощи JavaScript? 
1. Использовать JavaScript для изменения содержимого таблицы.
2. Обновлять данные с помощью AJAX или Fetch API.
3. Перерисовывать таблицу на основе новых данных.

Пример:

function updateTable(data) {
    let tableBody = document.getElementById('tableBody');
    tableBody.innerHTML = ''; // Очистка таблицы

    data.forEach(item => {
        let row = tableBody.insertRow();
        let cell1 = row.insertCell();
        let cell2 = row.insertCell();
        cell1.textContent = item.name;
        cell2.textContent = item.value;
    });
}

// Пример обновления данных
fetch('data.json')
    .then(response => response.json())
    .then(data => updateTable(data));
